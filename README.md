# Sentiment Analysis Web App

A Streamlit web application for sentiment analysis using BERT fine-tuned with LoRA.

## Features

- Real-time sentiment analysis (Negative/Neutral/Positive)
- Confidence scores and visualizations
- Pre-trained model using LoRA fine-tuning
- Clean, responsive UI

## Model Details

- Base Model: `bert-base-uncased`
- Fine-tuning: LoRA (Low-Rank Adaptation)
- Classes: 3 (Negative, Neutral, Positive)
- Training: Custom dataset
